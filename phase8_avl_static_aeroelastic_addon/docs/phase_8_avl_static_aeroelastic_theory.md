# Phase 8 — Static Aeroelastic Response Using AVL Spanload

This phase replaces the earlier ideal elliptic load with the AVL spanwise lift distribution.

The analysis chain is:

AVL spanload -> shear -> bending moment -> curvature through EI -> deflection -> coupling-induced twist through Kbt/GJ.

This is still a simplified static aeroelastic model. It does not yet include a detailed AVL section pitching-moment distribution about the elastic axis.
